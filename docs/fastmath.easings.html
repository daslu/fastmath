Save incanter graph

(defn save-graph
  [f params & opts]
  (let [fname (str "images/e/" (first opts) ".png")]
    (i/save (c/function-plot f 0.0 1.0 :y-label "easing value")
            (str "docs/" fname)
            :width 500
            :height 250)
    fname))

back-in

(back-in t)(back-in s t)

BackIn easing.

Parameter s (default: 1.70158) defines overshoot.

Examples

Usage

(m/approx (back-in 0.5) 6)
;;=> -0.087698

;; Test: ok.

Different overshot

(back-in 2.0 0.5)
;;=> -0.125

Plot

(save-graph back-in ...)

Plot, different s value.

elastic-in

(elastic-in t)(elastic-in amplitude period)

ElasticIn.

When called with t parameter, returns easing value (for amplitude=1.0 and period=0.3). When called with amplitude and period returns custom easing function.

Examples

Usage

(m/approx (elastic-in 0.2) 6)
;;=> -0.001953

;; Test: ok.

Create custom elastic easing

(m/approx ((elastic-in 1.0 0.1) 0.2) 6)
;;=> 0.003906

(save-graph (elastic-in 1.1 0.1) ...)

reflect

(reflect ease center)

Create in-out easing for given ease function and center.

Examples

Usage

(let [neasing (reflect (partial back-in 2.0) 0.2)]
  [(neasing 0.1) (neasing 0.5) (neasing 0.9)])
;;=> [-0.025 1.0390625 1.0203125]

For center=0.5 function returns regular in-out easing

(== (back-in-out 0.4) ((reflect back-in 0.5) 0.4))
;;=> true

;; Test: ok.

Reflect one easing in center=0.2 

(save-graph (reflect elastic-in-out 0.2) ...)
